# X.509 - X.509 证书

提供 X.509 证书的操作功能。

## 类

### X509Helper

X.509 证书帮助类，提供 X.509 证书的读取、验证和导出功能。

## 扩展方法

## 示例

### 加载证书

```csharp
// 加载无密码证书
var cert = X509Helper.LoadCertificate("certificate.cer");

// 加载有密码证书
var cert = X509Helper.LoadCertificate("certificate.pfx", "password123");
```

### 获取证书信息

```csharp
var cert = X509Helper.LoadCertificate("certificate.cer");

// 获取公钥
var publicKey = cert.GetPublicKey();
Console.WriteLine($"公钥: {publicKey}");

// 获取主题
var subject = cert.GetSubject();
Console.WriteLine($"主题: {subject}");

// 获取颁发者
var issuer = cert.GetIssuer();
Console.WriteLine($"颁发者: {issuer}");

// 获取过期日期
var expiration = cert.GetExpirationDate();
Console.WriteLine($"过期日期: {expiration}");

// 检查是否过期
var isExpired = cert.IsExpired();
Console.WriteLine(isExpired ? "证书已过期" : "证书有效");
```

## 特性

- 支持加载 X.509 证书文件
- 支持带密码的证书
- 支持获取证书公钥
- 支持获取证书主题和颁发者
- 支持获取证书过期日期
- 支持检查证书是否过期
